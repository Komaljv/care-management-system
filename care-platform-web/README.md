# care-platform-web

A Next.js frontend starter for the Care Management System.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run test`

## Run locally

1. Install dependencies: `npm install`
2. Start in development mode: `npm run dev`
3. Open `http://localhost:3000`

## Project layout

- `src/app` — app router pages and layouts
- `src/components` — reusable UI components

## Styling

This project uses Tailwind CSS for layout and styling.
